docker 빌드 시에 platform을 명시해주는 방법

```
docker build -t my-ubuntu:x86_64 --platform linux/amd64 .
docker run -it --rm my-ubuntu:x86_64 uname -m
```

docker-compose 으로 실행 시,

```
services:
  web:
    build: .
    command: uname -m
    platform: linux/amd64
```
